# Contracts: Email Classification Correction UI

**Feature**: 003-correction-ui
**Date**: 2025-11-22

---

## Purpose

This directory contains interface contracts for the Email Classification Correction UI. These contracts define the expected structure of API operations, component props, and event payloads without implementation details.

---

## Contract Files

1. **api-contracts.ts** - Supabase database operation interfaces
2. **component-contracts.ts** - Vue component prop and event type definitions
3. **service-contracts.ts** - Service layer function signatures

---

## Usage

These contracts serve as:
- Type-safe interfaces for implementation
- Documentation of expected behaviors
- Test fixtures for mocking
- Validation schemas for runtime checks
